Analysis Outputs
delta = score(condition) − score(blind) matched on (response_model, judge_model, prompt_id, rep)
⇅
Asymmetry Test
Mean delta for upward vs downward conditions. Tests whether the penalty for downward mis-attribution is larger in magnitude than the bonus for upward.
▦
Delta by Task Type
Attribution delta broken down by analytical, creative, and factual tasks. Tests whether bias is stronger where quality is more objectively judgeable.
◈
Delta by Judge Model
Attribution delta per judge model. Tests whether weaker judge models are more susceptible to prestige bias than stronger ones.
⇄
Ranking Inversions
Cases where a lower-prestige response attributed upward outscores a higher-prestige response attributed downward, for the same judge and prompt.
Statistical Tests
①
Wilcoxon Signed-Rank — Asymmetry
Tests whether |downward delta| > |upward delta| on matched pairs.

Why: Scores are integers on a 1–10 scale — not normally distributed, with many ties. Wilcoxon makes no normality assumption and handles ties correctly. Pairs are matched on (response_model, judge_model, prompt_id, rep) so each comparison controls for response quality. Reports p-value and rank-biserial r as effect size.
②
One-Sample Wilcoxon — Delta ≠ 0
Tests whether the mean attribution delta is significantly different from zero for each condition (true, upward, downward).

Why: Blind is the within-subjects baseline. A delta of zero means the attribution label had no effect. This test asks: is the observed shift real, or within noise? Non-parametric for the same reason as above.
③
Kruskal-Wallis — Group Differences
Tests whether the downward delta distribution differs across task types (analytical, creative, factual) and across judge models.

Why: Non-parametric alternative to one-way ANOVA. Used as an omnibus test before running pairwise comparisons — avoids inflating false positives by only running pairwise tests when the overall group difference is significant.
④
Mann-Whitney U — Pairwise + Bonferroni
Pairwise comparisons between task types and between judge models on downward delta. Bonferroni correction applied: α divided by number of pairs.

Why: With 3 task types there are 3 pairs; with 4 judges there are 6 pairs. Running multiple tests at α=0.05 inflates false positive rate. Bonferroni is conservative but appropriate at this sample size. Reports rank-biserial r alongside p-value.
Power caveat: At ~480 total judgments split across 4 conditions × 3 task types × 3 judges, per-cell n is approximately 20–40. The experiment is likely underpowered for detecting small effects. Results should be interpreted as preliminary signal, not confirmatory evidence. A full run with more models and prompts is required for significance claims.
